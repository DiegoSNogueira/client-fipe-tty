# client-fipe-tty

Consumindo a API da [Fipe do Deivid Fortuna](https://deividfortuna.github.io/fipe/) usando ruby.
## Tecnologia utilizada
* Ruby 2.6.5

## Gems utilizadas
1. gem 'rest-client'
1. gem 'tty-prompt'
1. gem 'tty-table'
